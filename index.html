MNI2TS

A utility for MEEG studies for identifying group-level activations, extracting relevant timeseries, performing t-contrasts & visualising.

Download this project as a .zip file Download this project as a tar.gz file

What is it?

It's a set of matlab functions [with a simple gui] for plotting source localised SPM12 MEEG data on meshes at group level, identifying peaks and extracting relevant timeseries. Performing t-contrasts between conditions and visualisation.

What does it do?

It loads up a template mesh and plots the group 'averaged' overlay for a given trial on top. From this, you can click an area of interest to find the coordinates of this 'peak'. There's a function that then finds each individual subjects' nearest source to that point [within a given radius]. You can then either reconstruct the data [time by trials] for the given identified vertices and store in a matrix, or call an spm built in function which extracts those vertices into a new LFP-SPM object for each individual dataset.

Why'd you make it?

Because I recently came to SPM from other softwares and I wanted to keep the same analysis approach that I'd used previously.

How to use?

For the gui: VSExtractor_gui

For a script: Click_to_extract_virtual_sensors
